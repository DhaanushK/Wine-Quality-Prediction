# Wine Quality Prediction using Deep Learning

## Overview
This project predicts wine quality using a hybrid deep learning model (MC-CNN + MLP).

## Tech Stack
- Python
- TensorFlow
- NumPy, Pandas
- Scikit-learn
- Matplotlib, Seaborn

## Workflow
1. Data preprocessing (StandardScaler, normalization)
2. Train-test split (80:20)
3. Model training with dropout (0.5)
4. Evaluation and visualization

## Results
- 87.6% accuracy (Red Wine)
- 90% accuracy (White Wine)

## Key Features
- Feature scaling
- Overfitting control
- Visualization of correlations
